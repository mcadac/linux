# Commands

1. Show what process is running on a specific port.

    **sudo netstat -lnpt | grep "puerto**
    
2. Show what is the service's port.

    **ps aux | grep "servicio"**
    
    **sudo netstat -lnpt | grep "proceso"**

3. Show memory use

   **sudo df -h**
